## 推奨アクション

### 優先度: 高（すぐに追加を検討）

1. **Field** - フォームの構築を劇的に改善する2025年新規コンポーネント
2. **Input Group** - 検索フィールドなどでよく使うパターン
3. **Button Group** - ボタンのグループ化を標準化
4. **Empty** - 空状態の表示を統一
5. **Spinner** - ローディング表示の標準化
6. **Data Table** - データ一覧表示の強力な選択肢
7. **Combobox** - 検索・選択UIの強化
8. **Date Picker** - 日付選択UIの改善
9. **Toast** - ユーザーフィードバックの標準化（Sonnerとの違いに注意）
10. **Chart** - データビジュアライゼーションが必要な場合

### 優先度: 中（必要に応じて追加）

1. **Item** - 多目的コンポーネント
2. **Kbd** - ショートカット表示に便利
3. **Native Select** - シンプルな選択UIが必要な場合
4. **Typography** - タイポグラフィの統一が必要な場合

## 注意事項

### Toastについて
- プロジェクトには既に **Sonner** が含まれています
- **Toast** は別の実装です（shadcn/uiの標準トーストコンポーネント）
- 両者の違いを理解した上で、どちらを使用するか決定してください

### Comboboxについて
- **Command** コンポーネントは既に存在します
- **Combobox** はCommandを使用した高レベルの実装です
- より簡単に使えるラッパーコンポーネントとして追加を検討してください

### Date Pickerについて
- **Calendar** コンポーネントは既に存在します
- **Date Picker** はCalendarを使用した高レベルの実装です
- 日付選択UIをより簡単に実装できます

## インストール方法

shadcn/uiのコンポーネントは以下のコマンドでインストールできます：

```bash
npx shadcn@latest add [component-name]
```

例：
```bash
npx shadcn@latest add field
npx shadcn@latest add button-group
npx shadcn@latest add spinner
npx shadcn@latest add data-table
npx shadcn@latest add combobox
npx shadcn@latest add date-picker
```
