## 統計情報

### 階層別統計
- **第1階層（製品）**: 1個
- **第2階層（Assy）**: 5個
- **第3階層（SubAssy）**: 17個
- **第4階層（SubSubAssy）**: 18個
- **第5階層（Module）**: 8個
- **第6階層（LeafProduct）**: 約60個

### タイプ別統計
- **Directory**: 約49個（製品1 + Assy5 + SubAssy17 + SubSubAssy18 + Module8）
- **LeafProduct**: 約60個
- **Document**: 約15個

### 特徴
- 複雑な6階層構造
- 各階層にDocument（図面・仕様書）が散在（Documentも罫線で表現）
- モジュール単位での部品管理（Module階層）
- 左右対称の構造（フィンガーモジュール左/右など）
- 実際の産業用ロボットアームの構成に近い
